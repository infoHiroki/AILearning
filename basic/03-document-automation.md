# 文書作成自動化

**所要時間**: 30分  
**レベル**: 基礎  
**前提知識**: [プロンプト基礎](02-prompt-basics.md)

## 学習目標

この講座を修了すると、以下ができるようになります：
- 議事録、メール、企画書などの文書を効率的に作成できる
- シンプルな指示から始めて、必要に応じてプロンプトを改良できる

---

## 基本原則：2段階アプローチ

### レベル1：シンプルに始める
最新のLLMは文脈を理解するため、**一言の指示**でも十分な結果が出ます。
まずはシンプルに始めましょう。

### レベル2：プロンプトを育てる
基本結果に満足できない場合、プロンプト基礎の知識を活かして改良します。

---

## 議事録作成

### レベル1：基本の使い方
```
議事録を作成してください。

[会議内容をそのまま貼り付け]
```

### レベル2：プロンプトを育てる
結果が物足りない場合：
```
以下の会議内容から議事録を作成してください。

[会議内容]

以下の形式で出力してください：
- 決定事項
- アクションアイテム（担当者・期限）
- 次回予定
```

### 実際の例
**入力**：
```
議事録を作成してください。

展示会の準備について話し合った。田中さんがブース設営、佐藤さんがパンフレット作成を担当。予算は50万円。来週火曜日に再度会議。
```

**出力例**：
```
【展示会準備 議事録】

■ 決定事項
- 予算：50万円
- 役割分担確定

■ アクションアイテム
- ブース設営：田中さん
- パンフレット作成：佐藤さん

■ 次回会議
- 日時：来週火曜日
```

## メール作成

### レベル1：基本の使い方
```
[状況]についてメールを作成してください。

相手：[相手の情報]
内容：[伝えたい内容]
```

### レベル2：プロンプトを育てる
```
以下の条件でビジネスメールを作成してください。

相手：[相手の詳細情報]
目的：[具体的な目的]
関係性：[初回/継続など]
トーン：[丁寧/親しみやすい等]

含めたい内容：
- [要点1]
- [要点2]
```

### よくあるパターン

#### お詫び
```
[問題内容]についてお詫びメールを作成してください。
```

#### アポイント依頼
```
[会社名]の[担当者名]に面談依頼メールを作成してください。
目的：[商品・サービス名]の提案
```

#### フォローアップ
```
昨日の会議のフォローアップメールを作成してください。
内容：[会議で話した内容]
```

## 企画書・提案書

### レベル1：基本の使い方
```
[テーマ]の企画書アウトラインを作成してください。

目的：[目的]
対象読者：[読者]
```

### レベル2：プロンプトを育てる
```
以下の企画書アウトラインを作成してください。

テーマ：[具体的なテーマ]
目的：[達成したい目標]
対象読者：[意思決定者の詳細]
想定ページ数：[ページ数]

重視すべきポイント：
- [ポイント1]
- [ポイント2]

各章に含めるべき要素も提案してください。
```

## プロンプト改良のコツ

### 結果が不十分な場合のチェックポイント

1. **情報不足**
   - 必要な情報をすべて含めているか？
   - 背景情報は十分か？

2. **出力形式**
   - 求める形式を明確に指定したか？
   - 文字数や構成に要求があるか？

3. **対象読者**
   - 誰に向けた文書かを明確にしたか？
   - 適切なトーンを指定したか？

### 改良の手順
1. **基本版で試す** → 結果を確認
2. **不足部分を特定** → 追加情報や指示を検討
3. **プロンプトを改良** → 再実行
4. **満足いく結果まで繰り返し**

---

## 注意事項

### 必ず人間がチェックすること
- **事実確認**：データや日付の正確性
- **内容の妥当性**：論理的な矛盾がないか
- **相手への配慮**：失礼な表現がないか
- **法的・倫理的問題**：コンプライアンス確認

### AIが苦手なこと
- 最新情報の反映
- 微妙なニュアンスの表現
- 法的責任を伴う最終判断
- 相手の感情の完全な理解
